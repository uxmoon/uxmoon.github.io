Use of pink

The lastminute.com pink is a fundamental part of our brand and it’s important for customer recognition that this colour is used throughout the site. It is equally important though, to ensure it isn’t misused or overused. As the colour is so closely linked with our brand image it is important that however and wherever pink is used on the site it has no negative connotations. Pink should only be associated with good things.

Primary button/call to action - takes the user forward further into the booking process.

Links/secondary buttons - but only if they that take the user forward or to a new page and is only the case if these items appear on a white background.

Highlights - important information which the user needs to be able to see at-a-glance.

Things like:

  • Total prices (when they appear on a white background)
  • Product/category icons
  • Number of offers found on a search
  • Pagination on a results page

Error messages - pink should never be associated with any problems or technical diificulties on our site. Error meassages are always red in colour. Read more detailed guidelines regarding the appearance of error messages on lastminute.com

Back links - any links that take the user backwards or that update the current page should not be pink.
