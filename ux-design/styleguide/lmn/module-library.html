Module library

The module library details the modules that can be used on the lastminute.com homepage. They are split into 4 categories. Inspirational, Inspirational / Deals, Deals & Functional. To help you understand the layout strategy, we've put together a template which details which modules should be used in which areas of the page. View layout strategy.

Below you can download photoshop files of the individual modules. If you'd like to download the full photoshop template including top nav, etc. you can also do that here. Download full photoshop template. Alternatively, download jpeg template.

Please note that we do not have licenses for all the images used in the templates so only use images you know have the correct licenses.

If none of the modules meet a specific need you have, contact Richard Spencer. A new module will then be designed & built.
