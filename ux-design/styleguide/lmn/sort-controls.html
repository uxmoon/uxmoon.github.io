Sort controls

Hotels

For the Hotels Path the Sort Controls only appear in the Search Results page above the hotel cards and below the TSH disclaimer in the content area.

Default

The default selection is "Our favorites" highlighted in magenta #ec008c. The "Distance from" and the dropdown for the landmarks is always displayed. The "Distance from" label is not clickable.
The width or lenght for the dropdown menu depends on the content and names of the landmarks.

Mouseover

On mouseover the link behaves like a regular link, the underline disappear upon user interaction with the link.

Distance from landmark

When the user selects a landmark from the dropdown the "Distance from" text is highlighted with a magenta color #ec008c.

Non-JS

When the user has a browser with JS disable the "Distance from" label and dropdown will not be shown.

This is deprecated. When the user has a browser with JS disabled the "Sort" button will appear so that the user can select a landmark

Flights

The sort controls in the Fligt Path will appear in the Search Results page above the flight cards with the "lowest price" sort selected by default.

X-Sell

In the X-Sell path the the "Distance from" label and dropdown will not be displayed.

Design breakdown

Colors
  • Default link color: #000
  • Highlight color: #ec008c
  • Pipe/Vertical divider: #999
Fonts and weights
  • Highlight link font-weight: bold
  • Default link font-weight: normal
  • Size font-size: 12pt
Visual Bug

There should not be any vertical divider or pipe to the right of the last link.
