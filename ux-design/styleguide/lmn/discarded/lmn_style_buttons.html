buttons & links

There are two button designs that should be used across the lastminute.com website. There is a primary and secondary button design. These are illustrated below in Fig 1. below.

Design notes:

  • 22px height
  • Varied widths
  • Centrally aligned text.
  • lowercase
  • 12px equivalent text (text should be scalable but this should be default)

The primary button should be used all the time for actions taking the customer forwards in the booking path. A secondary button will accompany this when there are two buttons relating to the same product/area that take the user to a different parts of the website.

For example, on the holidays search results there are two buttons next to each other. ‘check prices’ is the primary button & ‘view details’ the secondary. Check prices is the primary as it takes the user further into the booking process. The buttons should always sit next to each other with the primary button always on the right.
